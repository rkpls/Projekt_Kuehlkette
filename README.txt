README

Schritt 1: Plug-In's installieren

    Installiere die Plug-in's anhand der Anleitung_plugins.

Schritt 2: Programm ausführen

    Starte das Programm über den Button "run python file".

Schritt 3: Handhabung

    Wähle im Fenster deine Sprache über die Buttons oben rechts aus.
    Standarteinstellung ist Deutsch/DE.
    Weitere Sprachen sind Englisch/EN und Arabisch/AR.

    Wähle im Dropdownmenü die gewünschte ID aus.
